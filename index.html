Privacy Policy

Questa applicazione gestisce i dati esclusivamente in locale, direttamente sul dispositivo dell’utente, senza alcuna condivisione o trasferimento verso server esterni.

Dati raccolti

L’app salva i seguenti dati solo localmente:

  • Informazioni inserite dall’utente per la gestione del condominio (es. pulizie, interventi, note, date).
  • Nessun dato personale identificativo (email, nome utente, ecc.) viene raccolto o richiesto.

Uso dei dati

I dati sono utilizzati unicamente per fornire le funzionalità dell’app e migliorare l’esperienza d’uso.

Condivisione dei dati

I dati non vengono condivisi né trasmessi a terze parti, servizi cloud, o strumenti di analisi.

Conservazione dei dati

I dati sono memorizzati esclusivamente sul dispositivo dell’utente, tramite database SQLite. Disinstallando l’app tutti i dati vengono cancellati in modo permanente.

Diritti dell’utente

L’utente può in qualsiasi momento cancellare i dati dall’app o disinstallarla per eliminare completamente tutte le informazioni salvate.
